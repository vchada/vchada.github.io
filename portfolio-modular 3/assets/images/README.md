# Images Folder

Place your images in this folder:

- `profile-placeholder.svg` - Profile photo (replace with your actual photo)
- `project-*.png/jpg` - Project screenshots
- `blog-*.png/jpg` - Blog cover images
- `logo.svg` - Site logo (if needed)

## Recommended Sizes:
- Profile photo: 400x400px
- Project images: 800x600px
- Blog covers: 1200x630px
