# Learning Roadmap

## Agentic AI

- Build simple agents that can classify a request, choose a tool, execute an action, and explain the result.
- Add memory using a relational store first, then vector search for semantic recall.
- Practice safe agent design with approvals, audit logs, and bounded tool access.
- Compare LangChain4j and Spring AI patterns for Java-first agent applications.

## Full Stack Java

- Use Java 21, Spring Boot 3, Spring Security, Spring Data JPA, Flyway, and OpenAPI.
- Build React and TypeScript frontends with form validation, optimistic updates, and loading/error states.
- Add integration tests with Testcontainers for PostgreSQL, Redis, and Kafka.
- Package services with Docker and add GitHub Actions checks.

## Portfolio Goals

- Keep each project focused on one practical business use case.
- Write READMEs that explain the problem, architecture, learning goals, and next steps.
- Prefer small frequent commits that show steady progress.
- Avoid overclaiming. Every project should be clear about what is implemented and what is planned.
